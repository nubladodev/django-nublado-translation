django-nublado-translation


django-nublado-translation is a Django app for basic model translation. Fields to be translated in a model
are copied to a corresponding translation model and ...

Detailed documentation is in the "docs" directory.

Quick start
-----------

1. Add "django_nublado_translation" to your INSTALLED_APPS setting like this::

    INSTALLED_APPS = [
        ...,
        "django_nublado_translation",
    ]
